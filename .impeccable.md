## Design Context

### Users
Primary users are teachers doing high-frequency homework grading on their own computers. They need to move quickly between students, mark right or wrong with minimal friction, and trust the data because it feeds class analysis and follow-up teaching decisions.

### Brand Personality
Precise, steady, and efficient. The interface should feel like a dependable teacher's instrument panel: calm under pressure, fast to scan, and serious enough for daily use.

### Aesthetic Direction
Editorial utility. Use warm paper-tinted surfaces, dense information blocks, crisp type hierarchy, and restrained signal colors so the product feels like a modern grading ledger instead of a generic dashboard.

### Design Principles
1. Prioritize scan speed over decoration.
2. Make live grading status unmistakable at a glance.
3. Use color as signal, not ornament.
4. Keep every critical action close to the current context.
5. Preserve a calm visual rhythm during repetitive work.
